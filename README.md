# Spotify Data Analytics Dashboard 🎧

This project is a **Spotify Data Analytics Dashboard** built using **Python, Django, Pandas and Chart.js**.  
It analyzes a Spotify dataset and displays insights through interactive charts and visualizations.

## Features

- Dashboard showing total tracks, top genre, and top artist
- Top 10 genres visualization using bar charts
- Explicit vs Non-Explicit songs analysis
- Search songs or artists
- Artist detail page with popular songs
- Add songs to personal favorites library

## Technologies Used

- Python
- Django
- Pandas
- Chart.js
- HTML / CSS
- SQLite

## Dataset

The dashboard uses a Spotify dataset containing **114,000+ tracks** with information such as:

- Track name
- Artist
- Genre
- Popularity
- Explicit content
